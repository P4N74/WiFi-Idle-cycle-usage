WiFi-Idle-cycle-usage
=====================

Idle computers are used to calculate mathematical expressions which are broadcasted over Wi-Fi as requests.

Compilation
-----------

Requirements: CMake, libpcap

```
mkdir bin
cd bin
cmake ..
make
```

Usage
-----

```
./wifi-math interface
```
